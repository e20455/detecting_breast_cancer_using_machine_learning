# detecting_breast_cancer_using_machine_learning
Formed part of my final year dissertation project. I used the Breast Cancer Wisconsin (diagnostic) dataset from Kaggle to undergo data-preprocessing and extracted the most correlated features (data_preprocessing.py). 
I trained and evaluated various algorthims (evaluation.py) and then used optimisation technqiues to try increase their performance (optimisation.py). The model that performed best on the evaluation metrics was Logistic Regression with an accuracy score of 97%, this model was implemented into the user application (application.py). This application allows users to upload and test their own sets of features to see if they are likely malignant or benign. Also, the application features user authentication and stores features that are tested.
